Twitch TV

This is an Intermediate Front End Developer project from the FreeCodeCamp curriculum. The user stories to be completed are:

  • I can see whether Free Code Camp is currently streaming on Twitch.tv.
  • I can click the status output and be sent directly to the Free Code Camp's Twitch.tv channel.
  • If a Twitch user is currently streaming, I can see additional details about what they are streaming.

This was made using: Flexbox, SCSS, and native JS. No CSS or JavaScript frameworks or libraries were used. The code can be seen on Github.
